# Aircraft Readiness Analytics Simulation

Synthetic aircraft readiness analytics platform built using Java and interactive HTML/Chart.js visualization.

## Overview

This project demonstrates operational analytics concepts using a fully simulated dataset representing aircraft readiness, maintenance burden, utilization, and mission capability trends.

The application was designed to showcase:

* Object-oriented Java architecture
* Service-layer analytics calculations
* Interactive visualization generation
* Dashboard storytelling concepts
* Operational readiness modeling

All identifiers, metrics, trends, and maintenance events are synthetic and created solely for portfolio demonstration purposes.

## Features

* Fleet MC/NMC readiness calculations
* Maintenance burden analysis
* Aircraft utilization tracking
* MC vs NMC operational visualization
* Interactive scatter plot analysis
* Trend visualization across multiple reporting periods
* Java-generated HTML dashboard output

## Technology Stack

* Java
* Eclipse IDE
* HTML
* CSS
* Chart.js
* Object-Oriented Programming (OOP)

## Important Disclaimer

Synthetic Dataset | Portfolio Demonstration Only | No Operational Aircraft Data Utilized
